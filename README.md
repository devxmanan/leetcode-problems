## 🌟 Welcome to the Puzzle Playground

This repository is a curated collection of LeetCode solutions for essential algorithm problems, written in C++ and TypeScript. It is designed to help you explore clean implementations, follow common patterns, and sharpen your problem-solving skills.

---

## 🔥 What’s inside

- **Arrays** — efficient index tricks, product logic, maximizing values, and more.
- **Strings** — substring search, string counting, and pattern matching.
- **Trees** — recursive traversal, depth analysis, balancing, and transformation.

---

## ✨ Featured Solutions

| Folder | Problem | Language | Highlights |
|---|---|---|---|
| `Arrays` | `1. Two Sum` | C++ | Classic hash-table approach |
| `Arrays` | `238. Product of Array Except Self` | C++ | No division, linear time |
| `Strings` | `1967. Number of Strings That Appear as Substrings in Word.ts` | TypeScript | Clean substring filtering |
| `Trees` | `104. Maximum Depth of Binary Tree` | C++ | Recursive depth-first traversal |

---

## 🚀 How to use

1. Browse the folders by topic.
2. Read the problem title in the file name.
3. Open the solution and study the algorithm.
4. Run your own variations locally and modify the code.

---

## 🧠 Learning path

- Start with `Arrays` for index, sum, and rearrangement tasks.
- Move to `Strings` for pattern discovery and substring logic.
- Tackle `Trees` for recursive thinking and tree traversal methods.
